# DesafioDBServer

O programa foi feito na IDE Eclipse.
Foram utilizadas as bibliotecas JUnit 5 e Selenium para Java.
Foram utilizados o driver e o navegador Chrome para os testes.
Todos os processos do teste foram documentados.

Antes de iniciar o programa, mova a pasta "drivers" para a sua pasta "C:/" ou edite a linha 21 do código do arquivo "Teste.java".

Para iniciar o programa, execute o arquivo "Init.java";
Será instanciado um novo Objeto "Teste", e é a partir dele que o teste será executado.
Para mudar os dados à serem inseridos no formulário, edite o código à partir da linha 51 do arquivo "Teste.java".
Assim que o teste for finalizado, a página permanecerá aberta.
